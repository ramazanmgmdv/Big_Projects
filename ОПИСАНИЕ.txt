Добавление элемента:
лкм по кнопке "+" под таблицей, поочереди заполнить ячейки (Enter - завершение ввода в ячейки, переход к следующей)

Редактирование элемента:
пкм по нужной ячейке, заполнить ячейку (Enter - завершение ввода, Home - отмена)

Сохранение в файл:
лкм по кнопке "сохранить" (сохранение в том же файле)
или по кнопке "сохранить как", ввести путь к файлу (Enter - завершение ввода, Home - отмена)

Чтение из файла:
Указать путь к файлу в качестве параметра командной строки

Сортировка:
лкм по заголовку нужного столбца. Первый раз - сортировка по возрастанию, второй - по убыванию)

Вывод диаграммы:
пкм по заголовку нужного столбца
